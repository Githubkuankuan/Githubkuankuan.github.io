发布于: 2022-05-03 更新于: 2022-05-03 分类于: vue

一、是什么

diff 算法是一种通过同层的树节点进行比较的高效算法
其有两个特点：

  • 比较只会在同层级进行, 不会跨层级比较
  • 在diff比较的过程中，循环从两边向中间比较
    diff 算法在很多场景下都有应用，在 vue 中，作用于虚拟 dom 渲染成真实 dom 的新旧 VNode 节点比较
